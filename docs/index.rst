Bacanora: Optimized Tapis operations
====================================

.. image:: https://img.shields.io/badge/stability-under%20development-yellow.png
.. image:: https://img.shields.io/badge/python-3.6-brightgreen.png

Implements accelerated, failure-resilient versions of Tapis files operations,
plus workalike analogs of key Python `os.path` and `shutils` functions.
